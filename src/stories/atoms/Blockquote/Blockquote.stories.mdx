# Blockquote

## A container used for citing the title of a creative work

The Blockquote component is used to mark up the title of a cited creative work.
The reference may be in an abbreviated form according to context-appropriate
conventions related to citation metadata.

<ArgsTable story="Default" />

export const Template = (args) => <Blockquote {...args} />

export const content = 'Example blockquote text'
export const Example = () => content

## Color

You can set the color of the blockquote using the `color` prop. The color prop
accepts any valid CSS color value.
