# Grid

## A container used for spacing around elements

The Grid component is used to create a grid of elements. It is a wrapper around
the CSS Grid specification.

<ArgsTable story="Default" />

export const Template = (args) => <Grid {...args} />

export const content = 'One'
export const Example = () => content
